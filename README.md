# Food Lover Network 

## Key Features

- Users can browse and read food reviews submitted by others
- Registered users are able to submit new reviews with images and ratings
- Favorites functionality allows users to save and manage preferred reviews
- Search capability is available for finding reviews by food name or restaurant
- Authentication is provided via email/password and Google sign-in
- Each review has a dedicated details page with full information
- Users can edit or delete their own reviews
- The website is fully responsive and works on all devices
